# AutoRia Data Processing Tool

Java console application for fetching and processing car data from AutoRia API.

## Tech stack
- Java 17
- Maven
- Jackson
- HTTP Client

## Features
- Fetch data from AutoRia API
- Parse and validate JSON
- Business logic processing
- Save results to file

## How to run
1. Add API key in AutoRiaClient
2. mvn clean package
3. java -jar target/autoria-parser-1.0.0.jar
